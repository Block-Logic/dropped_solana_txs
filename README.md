# dropped_solana_txs
This contains log file data dumps of Solana transactions dropped from my validator. I am providing this information to help application developers who are curious to know why their TXs might be dropped.

Each TX was dropped from a block because it  `WouldExceedBlockMaxLimit` or `WouldExceedAccountMaxLimit`. You can will see the "hot accounts" in these files.

I will occasionally upload data dumps here in zip files.
